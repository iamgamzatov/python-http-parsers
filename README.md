# Python Web Parsers & HTTP Requests 🌐

Коллекция скриптов на Python для работы с веб-ресурсами, демонстрации базовых механизмов сетевых запросов (микро-парсинга) и взаимодействия с HTTP-методами.

## 📁 Структура проекта
* **`parser_GET.py`** — Скрипт, выполняющий **GET-запрос** с поисковыми параметрами. Демонстрирует работу с кодированием URL (`urllib.parse`), подстановкой заголовков `User-Agent` для обхода базовых блокировок и чтением HTML-кода.
* **`parser2_POST.py`** — Скрипт для работы с **POST-запросами**. Иллюстрирует передачу структурированных данных (payload) в теле запроса на сервер.
* **`download.py`** — Утилита для автоматического скачивания и сохранения контента/файлов из сети на локальный компьютер.

## 🛠 Стек технологий
* Python 3.x
* Urllib (встроенная библиотека для работы с HTTP)
* BeautifulSoup4 (парсинг и извлечение данных из HTML)

## 🚀 Как запустить
1. Клонировать репозиторий.
2. Установить зависимости: `pip install -r requirements.txt`
3. Запустить интересующий скрипт: `python parser_GET.py`
